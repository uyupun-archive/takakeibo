**Easily & Simply & Immediately**

### 環境構築

```bash
$ yarn install
$ cp .env.example .env
$ cp .env.local.example .env.local # その後、Firebaseの認証情報を追記する
$ yarn dev
```

### 本番環境

```bash
$ docker-compose up -d # 8080番ポートで待ち受け
```
